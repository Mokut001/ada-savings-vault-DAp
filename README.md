# ADA Savings Goal DApp

A simple Cardano DeFi DApp that helps users save ADA toward a target goal.

## Features
- Target-based ADA saving (e.g. 1000 ADA in 12 months)
- Monthly saving calculation
- Progress notifier
- Withdrawal when goal is reached

## Tech Stack
- Plutus (Haskell) smart contract
- React + TypeScript frontend
- Deployable on Vercel

## Run Frontend
cd dapp
npm install
npm run dev
